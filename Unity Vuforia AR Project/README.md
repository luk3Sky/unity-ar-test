# Unity Vuforia AR Project

An Augmented Reality (AR) Project to test Vuforia Engine in Unity.

### Features
 - Image Target Tracking

### Project Specs
 -  Unity Editor 2020.3.2f1
 -  Vuforia 9.8.5
